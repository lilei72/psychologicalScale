# psychologicalScale

### 开发计划

### 功能
##### 1.从MainActivity跳转到结果展示需要三个参数，(因子,得分,说明)
##### 2.还需根据结果给出说明和建议(说明是根据当前测试内容的结果来的，后面可能会新增子测试)

#####   放到questions中。
##### 4.提交方法中要计算当前表的得分情况,总表要计算GSI(详情看res/img.png),分表看总分(详情看img_1.png)

#####   第二个建议的话也是分为两种，中度以上症状就提醒就医，轻度的话就有10类了？
